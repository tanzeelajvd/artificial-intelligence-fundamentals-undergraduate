# Artificial Intelligence Fundamentals (Undergraduate)

This repository contains **complete undergraduate-level study material** for Artificial Intelligence, covering both **theoretical foundations** and **hands-on practical implementations**.

It is structured around two academic courses:

* **Artificial Intelligence (Theory)**
* **Artificial Intelligence Lab (Implementation)**

The goal of this repository is to provide a **clean, structured, and practical learning path** for understanding Artificial Intelligence from first principles through implementation.
